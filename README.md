# 個人知識與品味脈絡系統

個人知識與品味脈絡系統是一個由人工智能代理協助運作的個人脈絡系統，用來建立知識、聯想與品味。它的目標不是把網頁暫存起來，而是把文章、文件、網址與想法轉化成可反覆使用的知識、個人判斷、跨來源聯想與品味線索。

目前版本：v0.1.0。這是早期公開版本，先公開產品定位與後續封裝方向；完整安裝包、命令列工具與可重現示例仍在整理中。

## 這個系統做什麼

- 從公開網址或可讀文字擷取來源內容。
- 將來源整理成知識點、來源摘要與可檢查的引用基礎。
- 判斷內容對使用者的實際用途，而不是只做摘要。
- 連接多個來源，形成補底、加深、連接、修正與行動五類跨來源洞見。
- 以 Obsidian 為優先保存方式，保存長期可累積的知識庫。
- 產出面向閱讀與決策的知識成果頁，而不是人工智能任務紀錄。

## 適合誰

- 想把閱讀、研究、工作資料與生活觀察沉澱成第二大腦的人。
- 想讓人工智能幫忙做來源整理，但不想失去可追溯依據的人。
- 想把知識連到個人決策、品味、行動與長期項目的人。
- 想建立本地優先、可逐步公開封裝的個人知識工作流的人。

## 底層協作機制

本倉庫使用 Agent Handoff Kit 作為人工智能協作、跨對話接續、健康檢查與治理狀態保存的底層機制。它不是本產品的功能主體。

對使用者而言，產品主體是「個人知識與品味脈絡系統」：來源擷取、知識點、以 Obsidian 為優先保存方式的知識庫、跨來源聯想、品味線索與成果頁。Agent Handoff Kit 只負責讓人工智能代理在這個倉庫內可靠地接續工作、升級治理檔案、保存交接狀態。

## 未來安裝方向

公開版的正式安裝方式會以「個人知識與品味脈絡系統」自身的套件、命令列入口或代理指令為入口。Agent Handoff Kit 只會作為底層依賴或治理基座，不會被包裝成產品本體。

在完整套件出現前，請不要把本 README 當成最終安裝教學。當前倉庫主要供早期審閱產品定位與後續封裝方向。
