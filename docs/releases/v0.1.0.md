# v0.1.0｜公開定位糾偏後的產品起點

發布日期：2026-07-05

## 這次版本帶來甚麼

這是「意脈系統」的第一個公開倉庫版本。它先確立產品定位與後續封裝方向。

## 對用戶有甚麼直接好處

使用者可以先看清楚這個系統的目標：把文章、文件、網址與想法轉化成可反覆使用的知識、個人判斷、跨來源洞見與下一步行動。

## 用戶需要做甚麼

暫時不需要安裝。這一版還不是完整可安裝產品；請等待後續 package、CLI 或可重現示例。

## 安裝與升級

正式安裝入口尚未發布。未來入口應安裝或執行「意脈系統」本身；底層 AI 協作機制只會作為依賴或治理基座。

## 注意事項

本版本尚未包含完整安裝包、命令列工具或可重現示例。
