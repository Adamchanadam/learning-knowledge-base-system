# 公開文件

這裡放公眾可讀的「個人知識與品味脈絡系統」產品文件。公開文件必須遵守 [`../公開工作區ATION_BOUNDARY.md`](../公開工作區ATION_BOUNDARY.md)。

目前 README 只承擔產品定位、公開狀態與邊界說明。正式安裝、升級、命令列入口、套件 或部署文件，必須等產品自身入口明確後再新增，不得把底層協作工具的安裝頁當作產品安裝文件。

## 寫作邊界

公開文件可以描述產品用途、安裝、升級、操作流程、示例和限制；不可搬入 營運工作區的 `dev/`、`outputs/`、交接歷史、內部研發證據或私人資料。

可以簡短提及底層人工智能協作機制，但文件主體必須服務個人知識與品味脈絡系統的使用者。
